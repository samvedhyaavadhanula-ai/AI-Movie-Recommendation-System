# AI Movie Recommendation System

## Overview

This project is a simple AI-based movie recommendation system developed using Python.

## Features

* Accepts user genre preferences
* Uses content-based filtering
* Calculates match scores
* Recommends movies based on relevance

## Technologies Used

* Python
* VS Code
* Git & GitHub

## Example Input

Action, Sci-Fi

## Example Output

Avengers
Inception
Interstellar
